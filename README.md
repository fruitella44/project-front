# project-front
UI for CRUD backend


> Admin RPG panel allows:
* Create new account
* Delete account
* Change name, nickname, race, profession, status (banned/not banned)
* By pressing on "Counter per page" it reveals more columns

 Used: html/css, javascript, jquery(ajax), maven, tomcat, jdk-17
